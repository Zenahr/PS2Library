1. Prepare your games folder

PS2Library currently supports only one folder structure. To use the program, your games folder needs to be structured like so:

/Games
/Games/<A Game>/
/Games/<A Game>/some-title.iso (or .bin, ...)
/Games/<A Game>/some-gibberish.png (or .jpg, .webp, ...)

An example of this:

.............................................................
.............................................................
├───10.000 Bullets
│       10,000_Bullets_PAL_Cover.jpg
│       10.000 Bullets.iso
│
├───Aeon Flux
│       512073-aeon-flux-playstation-2-front-cover.jpg
│       Aeon Flux.iso
│
├───Alias
│       412FVV72FFL._SY445_.jpg
│       Alias.iso
│
├───Beyond Good & Evil
│       Beyond Good & Evil.iso
│       download.jpg
│
├───Clock Tower 3
│       20859-clock-tower-3-playstation-2-front-cover.jpg
│       Clock Tower 3.iso
.............................................................
.............................................................

Sidenote: It doesn't matter how the files inside the individual game folders are named. PS2Library uses the game folder name to derive the display name. Similarly, the box art image can be named anything and doesn't have to be named like the game at all.
For © concerns PS2Library does not allow downloading cover images. However, you can use google (search for "ps2 cover <game name>") to find covers for your game.

Once you've added a cover image to all of your game folders, proceed with the next step.

2. Set up the path to your PCSX2 executable and games folder inside the config.ini file                                     

For that, open the file in any text editor and copy/paste the respective paths.

3. Open PS2Library.exe to start the frontend
